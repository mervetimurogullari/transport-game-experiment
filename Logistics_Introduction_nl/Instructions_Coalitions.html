Er is een premie van 9000 euro beschikbaar voor bedrijven die een coalitie (een samenwerking) vormen om samen de grotere vrachtwagen te gebruiken. Wij leggen hier in detail uit welke coalities gevormd kunnen worden.

Als er een coalitie gevormd wordt, moeten de deelnemers aan de coalitie beslissen hoe zij de 9000 euro verdelen. Deelnemers die niet bij de coalitie horen moeten hun eigen vrachtwagen gebruiken en krijgen dus ook geen deel van de premie.

Hieronder staan alle mogelijkheden voor een coalitie die voldoende is om de premie te krijgen:

    • {{ endif }}

    {{ if 'AC' in possible_coalitions_all }}
  • Bedrijf A en Bedrijf C kunnen een AC-coalitie vormen die (4 + 2) 6 ton vracht vervoert
    • {{ endif }}

    {{ if 'BC' in possible_coalitions_all }}
  • Bedrijf B en bedrijf C kunnen een BC-coalitie vormen die (3 + 2) 5 ton vracht vervoert
    • {{ endif }}

    {{ if 'ABC' in possible_coalitions_all }}
  • Bedrijf A, Bedrijf B en Bedrijf C kunnen een ABC-coalitie vormen die (4 + 3 + 2) 9 ton vracht vervoert
    • {{ endif }}

Let op: U moet ervoor zorgen dat u de best mogelijke deal krijgt voor uw eigen bedrijf. Welk bedrijf u vertegenwoordigt wordt willekeurig bepaald als alle deelnemers aan het onderzoek er zijn.
